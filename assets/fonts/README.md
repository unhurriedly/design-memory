# Offline fonts

These files are bundled so the Design Memory interface and component previews work without a network connection.

| Directory | Families | Original source |
| --- | --- | --- |
| `misans/` | MiSans, Outfit | MiniMax public site assets |
| `noto/` | Noto Sans SC, Noto Serif SC | Google Fonts |
| `benshih/` | Roobert, Acorn | Ben Shih public site assets |

The font files are runtime assets. Component code shown in the library uses paths relative to the project root; standalone demos under `components/` use paths relative to their own directories.
